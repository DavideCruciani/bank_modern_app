## Setup del Progetto

1. Clona la repository sul tuo computer:

```bash
git clone git@github.com:DavideCruciani/bank_modern_app.git
```

2. Installa le dipendenze necessarie:

```bash
npm install
```

3. Avvia il server di sviluppo:

```bash
npm run dev
```
